## 題目說明
反轉單向鏈結串列

## 解題思路
### 鏈結轉向：第一步
理解原本鏈結陣列長相，本來的 linked List 長這樣:
```
2 -> 3
```
每個list只知道自己，以及自己的下一個list是誰，所以只知道cur為2，cur.next為3

### 鏈結轉向：第二步
為了反轉方向，首先要斬斷2 => 3的關係，但是一旦斬斷關係，就無法再找回，故先用一個暫時變數來儲存
```
let nextTemp = cur.next
```
### 鏈結轉向：第三步
現在的狀態長這樣
```
<- 2  3
```
所以現在cur.next其實是指向原本的前一個node，這裡先稱之為prev，所以調整為這樣
```
cur.next = prev
```
### 鏈結轉向：第四步
現在要把箭頭移位置了，原本cur在2，現在要讓箭頭cur在3，這樣cur.next就會在2，讓鏈結位置變成這樣
```
2 <- 3
```
```
prev = cur
cur = nextTemp
```
### 鏈結轉向：第五步
已經完成箭頭的轉向，並放置在正確位置，最後就只要將prev傳出去即可
### 條件設定
一旦cur不存在即代表到底了
