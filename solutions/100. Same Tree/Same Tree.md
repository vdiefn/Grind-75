## 題目說明
給定兩棵二元樹的根節點 p 與 q，請撰寫一個函式來檢查這兩棵樹是否相同。
- 若兩棵二元樹在結構上完全一致，且每個對應節點的數值也相同，則視為相同的樹。

## 解題思路
### 方法一：遞迴解
1. 三個重要判斷：
    - p 和 q 都是空 -> 回傳 true
    - p 和 q 有一個是空的 -> 回傳false
    - p 和 q 的值不一致 -> 回傳false
2. 接著才是遞迴進行檢查左子樹和右子樹是否一致
### 方法二：BFS(廣度優先搜尋)
1. 一開始先將node放到一組空陣列(queue)中
2. 使用while迴圈，只要該陣列的長度>0，就持續進行迴圈
3. 將陣列的值使用shift取出，做三個重要判斷：
    - p 和 q 都是空 -> 繼續進行下一輪判斷
    - p 和 q 有一個是空的 -> 回傳false
    - p 和 q 的值不一致 -> 回傳false
4. 現在queue是空的，所以分別將左子樹和右子樹push進去
5. 最後while跑完沒有提早結束才會回傳true
