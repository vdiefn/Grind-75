## 題目說明
這題是經典的演算法問題，通常稱為 「爬樓梯問題」（Climbing Stairs）。本質上它是一個 費波那契數列（Fibonacci sequence） 的變體。
假設我們要爬到第 $n$ 階：如果要爬到第 n 階，最後一步只有兩種可能：
- 從第 n-1 階往上走 1 步
- 從第 n-2 階往上走 2 步。

因此，到達第 n 階的方法總數，就是「到達 n-1 階的方法數」加上「到達 n-2 階的方法數」。

## 解題思路
### 方法一、使用遞迴
因為到達第n階的方法是 n-1 階 + n-2 階，所以可以使用遞迴一值計算這兩個數字相加
```
var climbStairs = function(n) {
  if(n <= 2) return n
  if(n > 2) {
    return climbStairs(n-1)+climbStairs(n-2)
  }
};
```
但是這段程式碼在處理較大的 n（例如 n = 44）時，會遇到嚴重的效能問題(timeout)，這主要是因為「重複計算」過多。
### 方法二、使用迴圈
每一次都要記住前兩次的數字，因此每一個階段數字都只要記憶一次
- 這樣時間複雜度是 O(n)
- 因為只用了幾個變數，所以空間複雜度是 O(1)
