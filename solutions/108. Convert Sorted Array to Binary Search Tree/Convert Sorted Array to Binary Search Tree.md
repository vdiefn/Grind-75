## 題目說明
這道題目要求將一個由小到大排序好的整數陣列 nums，轉換成一棵高度平衡 (height-balanced) 的二元搜尋樹 (Binary Search Tree, BST)。
> 高度平衡 (Height-Balanced)：這意味著對於樹中的任何一個節點，其左子樹與右子樹的高度差不能超過 1。

## 解題思路
1. 二元搜尋樹（BST）的結構特性：根節點左側的數值都比較小，右側的數值都比較大
2. 解題步驟
    - 選取中心，找出中間值的index
    - 將原本的陣列拆成左半邊和右半邊
    - 使用遞迴一直尋找，直到達到遞迴終止條件 leftIndex > rightIndex
